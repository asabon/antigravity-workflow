# Project-Specific Customizations

このディレクトリ (`.agent/custom/`) は、プロジェクト固有のルールや指示を記述するための場所です。

## 役割
- **Core 規約のオーバーライド**: `.agent/rules/` にある汎用的なルールよりも、ここにあるファイルの内容が AI エージェントによって優先されます。
- **技術スタック固有の指示**: Android 固有のビルド手順、特定のライブラリの使用ルールなど、このプロジェクト専用のコンテキストをここに記述してください。

## 同期について
- このディレクトリ配下のファイルは、基盤リポジトリ (`antigravity-workflow`) からの `/sync-workflow` による**同期対象から除外されます。**
- つまり、ここに書いた内容は基盤の更新によって上書きされることはありません。

## 活用例
1. `android_rules.md`: Android 開発固有の規約（Compose 活用方針など）
2. `api_spec.md`: このプロジェクト独自の API 設計ガイドライン
3. `ignore_rules.md`: 特定の監視対象外にしたいファイルへの指示

---
> [!TIP]
> AI エージェントはセッション開始時にこのディレクトリを自律的にスキャンします。新しい規約を追加した際は、その旨を AI に伝えるか、一旦セッションをリセットすることで確実に反映されます。
